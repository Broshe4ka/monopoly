# Next.js app

Check deployments

### Running Locally

1. Install dependencies: `npm i`
2. Start the dev server: `npm run dev`
3. Check browser in `localhost:3000`

### Build project

To get a project build: `npm run build`

For more information, see: [NextJS](https://nextjs.org)
